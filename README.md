# ProyectoFinalUnidad1

Descripción del Proyecto - Tesoros de Canadá

Este proyecto consiste en el desarrollo de un sitio web informativo llamado Tesoros de Canadá, enfocado en mostrar información sobre la provincia de Alberta y uno de sus destinos más representativos: Banff.

El sitio cuenta con dos páginas principales:

- Home (index): información general sobre Alberta
- Banff: sección específica sobre el Parque Nacional Banff

# Obejtivo

Diseñar una página web visualmente atractiva y organizada, aplicando conocimientos básicos de HTML, CSS y Bootstrap, con el fin de presentar información turística de manera clara e interactiva. Siguiendo el diseño del mockup quen os proporcionó la actividad en el Aula Virtual de la universidad donde está la asignatura del Diplomado en Desarrollo Web.

# Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome (para íconos)

# Estructura del sitio

Página principal (Home)

1. Barra de navegación
Permite al usuario desplazarse entre las páginas del sitio.

2. Carrusel de imágenes
Implementado con Bootstrap para mostrar imágenes representativas de Canadá.

3. Sección 1
Contiene:

* Un párrafo alineado a la izquierda
* Una imagen en el centro
* Un párrafo a la derecha

Diseñada utilizando el sistema de grid de Bootstrap.

# Sección 2
Incluye:

* Una lista de los tres lugares más lindos de Banff
* Un párrafo descriptivo
* Una tabla con información de los mismo lugares que muestra la lista

Características: Es un detalle que quise añadir para que la página se viera más atractiva

* Efecto hover en la lista (color rojo pastel)
* Ícono de hoja de maple para personalización visual

# Sección 3
Contiene:

* Un párrafo centrado
* Un botón que redirecciona a la página de Banff

Interacción:
* Efecto hover que oscurece el botón para resaltar su importancia

# Sección 4
Incluye:

* Un párrafo
* Un video relacionado con Banff

# Footer
Dividido en tres partes:

Izquierda: nombre de la creadora
Centro: redes sociales con íconos
Derecha: enlaces de interés sobre Canadá

Interacciones:
* Efecto hover en íconos y enlaces (color azul)


# Página Banff

Contiene una estructura similar a la página principal, pero con una sección adicional:

Sección destacada:

* Texto con el título “Conoce Banff”
* Dos imágenes alineadas a la derecha
* Diseño centrado y equilibrado visualmente


# Características de diseño

* Uso de Flexbox y Bootstrap Grid para la organización del contenido
* Efectos hover para mejorar la interacción del usuario
* Diseño responsive adaptable a diferentes dispositivos 
* Uso de íconos para mejorar la experiencia visual
* Aplicación de estilos personalizados con CSS


# Funcionalidades

* Navegación entre páginas
* Carrusel interactivo
* Botón con redirección
* Efectos visuales al pasar el mouse
* Contenido organizado en secciones


# Aprendizajes

Durante el desarrollo de este proyecto se aprendió:

* Uso de etiquetas HTML (listas, tablas, enlaces)
* Aplicación de estilos con CSS
* Uso de Bootstrap para diseño responsive
* Manejo de estructuras con container, row y col
* Implementación de efectos visuales
